# Развернуть бота на бесплатном сервере (почти без ваших действий)

Бот будет работать **24/7 в интернете**, не на вашем компьютере. Есть несколько бесплатных вариантов (без банковской карты):

---

## Вариант 1: Pella.app (рекомендуется, если JustRunMy не подошёл)

**https://pella.app** — бесплатно, без карты, 24/7, без строгой проверки email.

### Шаги:

1. Откройте **https://pella.app** → **Sign Up** (можно через Google/GitHub)
2. **New App** → выберите **Python** → **Upload ZIP**
3. Загрузите архив **бот_для_сервера.zip** (создайте через `Собрать_архив_для_сервера.bat`)
4. В разделе **Environment Variables** добавьте:
   - `BOT_TOKEN` = `8247551271:AAHbn5fEWky9BTna3bfKrW1GwjV1TrbiH-o`
   - `ADMIN_IDS` = `1163170314`
5. Команда запуска: **`python main.py`**
6. **Deploy** → бот запустится

## Вариант 2: JustRunMy.App

**https://justrunmy.app** — бесплатно, без карты, 24/7.

---

## Шаг 1. Регистрация (1 минута)

1. Откройте в браузере: **https://justrunmy.app**
2. Нажмите **«Start for free»** / **«Host your bot free»**
3. Зарегистрируйтесь (email или через Google/GitHub)

---

## Шаг 2. Создать проект и загрузить бота

1. В панели нажмите **«New»** / **«Create»** (новый бот/проект)
2. Выберите способ: **Zip Upload**
3. Подготовьте архив:
   - Откройте папку с ботом (`при скачивалка`)
   - Выделите **все файлы и папки**, кроме:
     - папки **`data`** (её не включать)
     - папок **`__pycache__`** (не включать)
     - файла **`.env`** (не включать — токен укажете на сайте)
   - Создайте ZIP-архив из выделенного (ПКМ → «Отправить» → «Сжатая ZIP-папка» или через 7-Zip/WinRAR)
4. Загрузите этот ZIP на JustRunMy.App (кнопка загрузки в интерфейсе)

---

## Шаг 3. Указать переменные (токен и ID админа)

В настройках проекта найдите раздел **Environment** / **Variables** / **Secrets** и добавьте:

| Имя переменной | Значение |
|----------------|----------|
| `BOT_TOKEN` | Ваш токен бота (из .env: `8247551271:AAHbn5fEWky9BTna3bfKrW1GwjV1TrbiH-o`) |
| `ADMIN_IDS` | Ваш Telegram ID: `1163170314` |

Если сервис просит **TELEGRAM_BOT_TOKEN** вместо BOT_TOKEN — используйте это имя, бот его тоже понимает.

---

## Шаг 4. Команда запуска

В настройках запуска укажите команду:

```text
python main.py
```

Если сервис сам ставит зависимости по `requirements.txt`, ничего больше не нужно. Если есть поле «Build command», можно оставить пустым или указать:

```text
pip install -r requirements.txt
```

Затем **Start** / **Deploy** — бот должен запуститься и быть онлайн 24/7.

---

## Если что-то не так

- **Логи** — смотрите в панели раздел Logs, там видно ошибки.
- **FFmpeg** — на сервере может не быть FFmpeg; тогда скачивание видео может не работать, пока на хостинге не установят FFmpeg. Текст и админка будут работать.
- **Помощь JustRunMy.App**: на сайте есть раздел с инструкциями и поддержкой.

После этих шагов бот работает в облаке, ваш компьютер может быть выключен.

## Вариант 3: Railway.app (если нужен GitHub)

**https://railway.app** — $5 бесплатного кредита в месяц (хватит надолго).

**Требуется:** аккаунт GitHub (бесплатно).

### Шаги:

1. Зарегистрируйтесь на **https://railway.app** (можно через GitHub)
2. **New Project** → **Deploy from GitHub repo**
3. Если проекта нет на GitHub:
   - Создайте репозиторий на **https://github.com** (New repository)
   - Загрузите туда файлы бота (кнопка "uploading an existing file")
   - Не загружайте папку `data`, файл `.env`, папки `__pycache__`
4. В Railway выберите ваш репозиторий → **Deploy**
5. В настройках проекта → **Variables** добавьте:
   - `BOT_TOKEN` = `8247551271:AAHbn5fEWky9BTna3bfKrW1GwjV1TrbiH-o`
   - `ADMIN_IDS` = `1163170314`
6. Railway сам запустит бота по `main.py`
